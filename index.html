What this project demonstrates

채용 담당자가 빠르게 볼 수 있도록 문제정의, 데이터 처리, 시각화, 모델링, 의사결정 활용 순서로 정리.

Business question framing

“카페가 어디에 많은가”에서 “어떤 상권이 강하고, 무엇이 매출을 설명하는가”로 질문을 확장.

Geospatial visualization

전국 카페 매장과 주요 브랜드의 공간 분포를 지도 기반으로 비교.

Model-based validation

서울시 상권 프로파일링 데이터를 활용해 매출 상위권 영향요인을 Random Forest로 탐색.

Analytical flow

포트폴리오에서는 화려한 결과보다 “실제 업무에서 어떻게 문제를 구조화하는지”가 더 중요함.

Data foundation

  • 공공데이터포털 상가업소정보
  • 서울시 상권 프로파일링 데이터
  • 카페 업종 필터링 및 브랜드명 정제
  • 좌표 기반 지도 시각화 데이터 구성

Business interpretation

  • 전국 카페 및 주요 브랜드 입지 커버리지 비교
  • 서울 주요 상권의 매출 규모 비교
  • 고매출 상권과 비고매출 상권의 변수 차이 확인
  • 입지·상권·고객 프로파일 기반 의사결정 가능성 제시
Portfolio takeaway. 이 프로젝트는 오래된 수업 발표자료를 그대로 보여주는 자료가 아니라, 공개 데이터 기반 공간분석을 비즈니스 문제정의와 의사결정 지원 관점으로 재구성한 사례다.
